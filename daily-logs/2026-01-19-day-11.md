### 1. Automation as a Discipline
- Automation is an umbrella term covering:
  - Robotic Process Automation (RPA)
  - API automation
  - Test automation
  - Artificial Intelligence (AI)
- Core definition:
  > If activities inside a machine occur without human intervention, it is automation.

---

---

### 3. UiPath Core Platform Components
High-level understanding of how the system works:

- **Studio**  
  Designs and builds automation workflows

- **Robot + Assistant**  
  Executes automations (attended / unattended)

- **Orchestrator**  
  Centralized control for:
  - Publishing processes
  - Managing robots and licenses
  - Running jobs
  - Monitoring execution

Key insight:
> Automation in UiPath is a **distributed system**, not a single tool.
