## üß† Key Clarifications Gained

- Publishing is **not execution** ‚Äî it is packaging + distribution
- Orchestrator is the **control plane**, not a build tool
- Studio Web and Studio Desktop have **overlapping but not identical capabilities**

  - Execution target (attended vs unattended)
